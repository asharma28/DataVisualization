A journey through history of soccer

Soccer as a game is played all across the world.Purpose of this data visualization is to explore theÂ International Football results dataset and understand the evolution of game. This dataset lists all the international soccer games (as in games opposing countries or nations) from 1872 to early 2018. In this project focus is to do analysis on the data and find out 1. How the game as evolved in terms of goals ,scored ,matches. 2. What are the top team that played this game over years. Data Set : https://www.kaggle.com/martj42/international-football-results-from-1872-to-2017

Hybrid Structure
This project follows the interactive slideshow hybrid structure. Users are taken through a series of slides, or scenes. While in each scene, users are able to mouseover each state to dig into the details they may be interested in.

Scene Utilization Based on slideshow structure this project utilizes the keyboard arrow keys to navigate scenes as depicted by the author. Scenes follow order in the slideshow as author drills down into details. There are three scenes: A welcome scene, an "About This Project" , a data vizualization of number of games highlighting number of games played over history of the game. a data vizualization of average climate temperature in the USA during 2013, and a credits page. The geological map scenes are intended to show average land temperature provided by Kaggle's open source data set "GlobalLandTemperaturesByState." Annotations and tooltip
The annotation is represented by the two different colors of the axis used in the multiple line graph red from blue drawing attention of the user.

Triggers The mouse click event on the legend Total matches or Total goals results in toggle action for the line graph. The mouse click event acts as a trigger to faciliate back and forward navigation in this visualization presentation.

Parameters This section is hyper linked to the previous and next section. The name of the target section is passed as an input paramter to the buttons and navgiation bar items.
