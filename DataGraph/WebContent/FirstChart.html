User Interface Events

The navigation buttons at the top this page represent the User Interface events. User interacts by clicking on the button or menu items to navigate to various part of this visualization story.Also while mouse hovering tool tip is displayed highlighting details and drawing user attention

Annotations and tooltip

The annotation is represented by the tooltip when mouse is hovered over the bars in the chart. Also while hovering mouse over graph color of the highlighed bar changes to red from blue drawing attention of the user.

Triggers

The mouse click event acts as a trigger to faciliate back and forward navigation in this visualization presentation. The mouse hover (mouseover) event is a trigger for the tooltip in the bar chart.
