Summary
This visualzation slide deck is highlighting number of games played over history of the game against goals scored.This graph is utilzing Multiple line graph in v4.Clearly user can see as number of games increased it resulted in more number of goals score.

User Interface Events
The navigation buttons at the top and the Next button at the bottom of this page represent the User Interface events. User can toggle line graph to see only total matches line graph or total goals per decade line graph or both

Annotations and tooltip
The annotation is represented by the two different colors of the axis used in the multiple line graph red from blue drawing attention of the user.

Triggers The mouse click event on the legend Total matches or Total goals results in toggle action for the line graph. The mouse click event acts as a trigger to faciliate back and forward navigation in this visualization presentation.

Parameters This section is hyper linked to the previous and next section. The name of the target section is passed as an input paramter to the buttons and navgiation bar items. The mouse click event to toggle the line graphs is utlizing the parameters in this.
