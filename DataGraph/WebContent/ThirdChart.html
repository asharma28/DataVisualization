Summary
This visualzation slide deck is highlighting top 10 soccer playing nations over history of the game.Most of the game have been played by these teams. Overall dataset was explored and used to build this bar chart. This bar chart highlights Sweden as nation which has played most number for games. One thing to note in this visualzation is that there is not much difference between number of games played by top 10 nations.

User Interface Events
The navigation buttons at the top and the Next button at the bottom of this page represent the User Interface events. User interacts by clicking on the button or menu items to navigate to various part of this visualization story.Also while mouse hovering tool tip is displayed highlighting details and drawing user attention

Annotations and tooltip
The annotation is represented by the tooltip when mouse is hovered over the bars in the chart. Also while hovering mouse over graph color of the highlighed bar changes to red from blue drwaing attention of the user.

Triggers The mouse click event acts as a trigger to faciliate back and forward navigation in this visualization presentation. The mouse hover (mouseover) event is a trigger for the tooltip in the bar chart.

Parameters This section is hyper linked to the previous and next section. The name of the target section is passed as an input paramter to the buttons and navgiation bar items.
